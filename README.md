# DevHub
**PersonalHub** — Dashboard personal todo-en-uno con gestión de finanzas, tareas, calendario y un hub de recursos de desarrollo. Construido con Astro + React + Tailwind, UI glassmorphism con estética NVIDIA. Persistencia 100% localStorage, sin backend. Incluye DevBot, un copiloto con IA que analiza tus datos y ejecuta acciones.
